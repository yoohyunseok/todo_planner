# 한국어 일정 관리 플래너

마감일, 대시보드, 달력 뷰가 있는 순수 HTML/CSS/JS 단일 파일 플래너입니다.
Vercel로 배포되며 Supabase 데이터베이스에 데이터를 저장합니다.

---

## 실행 방법

### 온라인 (배포 버전)

Vercel 배포 URL로 접속하면 바로 사용 가능합니다. 설치 불필요.

### 로컬 실행

`todo_list_pdf.html` 파일을 브라우저로 열면 됩니다. 빌드 과정 없음.

---

## 주요 기능

### 할 일 관리
- 할 일 추가 / 수정 / 삭제
- 카테고리 분류: **업무 · 개인 · 공부**
- 텍스트 입력 시 카테고리 자동 분류 (키워드 기반)
- 마감일 설정 (선택 사항)

### 마감일 배지
| 상태 | 표시 |
|------|------|
| 기한 초과 | 🔴 `D+N 지남` |
| 오늘 마감 | 🟡 `오늘 마감` |
| 3일 이내 | 🟡 `D-N` |
| 여유 있음 | 🔵 `D-N` |

### 대시보드
- 전체 할 일 수 · 오늘 완료 · 마감 임박 · 지연 중 통계
- 카테고리별 완료율 진행 바

### 달력 뷰
- 월별 그리드에서 마감일 한눈에 확인
- 날짜 클릭 시 해당 일정만 필터링
- ‹ 오늘 › 버튼으로 월 이동

### 정렬 / 필터 / 일괄 삭제
- 정렬: 생성순 · 마감순 · 카테고리순 (정렬 설정은 localStorage 유지)
- 필터: 전체 · 업무 · 개인 · 공부
- 체크박스로 여러 항목 선택 후 일괄 삭제

---

## 데이터 저장

**Supabase** 데이터베이스에 저장됩니다. 어떤 기기와 브라우저에서 접속하더라도 동일한 데이터를 유지합니다.

### tasks 테이블 스키마

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | TEXT | 고유 식별자 (UUID) |
| `text` | TEXT | 할 일 내용 (최대 200자) |
| `category` | TEXT | 업무 / 개인 / 공부 |
| `completed` | BOOLEAN | 완료 여부 |
| `created_at` | TIMESTAMPTZ | 생성 일시 |
| `deadline` | DATE | 마감일 (선택) |

---

## 기술 스택

- **언어**: 순수 HTML / CSS / JavaScript
- **데이터베이스**: Supabase (PostgreSQL)
- **배포**: Vercel
- **의존성**: 없음 (외부 라이브러리 미사용, Supabase REST API 직접 호출)
- **데스크톱 전용** (권장 뷰포트: 1024px 이상)
